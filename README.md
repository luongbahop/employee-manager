# Employee management

### Technologies
#### Back-end
- [NestJs] 
- [TypeORM]
- [MySQL]

#### Web app
- [ReactJs] 
- [Redux-toolkit]
- [Ant-Design]

### Start app locally

```bash

git clone https://github.com/luongbahop/employee-manager.git

cd employee-manager

npm install

npm run install-all

npm run start

```
